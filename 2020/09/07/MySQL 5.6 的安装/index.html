MySQL 5.6的安装（zip版）

Author: Gao Zhiqiang Date: September 7, 2020  20:37:53 Category: Mysql 5.6

Mysql 5.6 安装步骤（zip版）

  • 下载安装包（MySQL-5.6.zip）

可根据系统选择 32位|64位 官网 www.mysql.com

  • 解压到一个磁盘目录 （如F:)
  • 为了方便把文件夹命名为（mysql-5.6）
  • 打开 我的电脑 → 属性 → 高级 → 环境变量
  • 在 Path 内将MySQL文件夹下的 bin 文件夹所在的目录路径添加到环境变量值 （f:\mysql-5.6\bin)
  • 然后在mysql-5.6文件夹下将my-default.ini文件复制一份并重命名为my.ini
  • 打开my.ini修改配置文件

修改安装目录|mysql数据库的数据存放目录

  • 以管理员身份运行cmd（！！！）
  • 进入mysql-5.6\bin
  • 输入 mysqld –initialize-insecure –user=mysql 回车
  • 输入 mysqld install 回车
  • 输入 net start mysql 启动mysql服务 （start 启动|stop 停止）
  • mysql -u root -p 回车 出现 enter passwore： ，提示输入密码时，由于刚安装所以直接回车
  • 输入 show databases; 查看默认安装数据库
  • 安装结束
